TeamCity 설정 1단계 06

Teamcity는 베타서비스인 cloud 무료 버전이다.

6. TeamCity Job 설정 1

  1. Create Pipeline을 선택하면 writerside project 계정을 선택한다.

  2. Job 1 선택

  3. Steps -> SCRIPT 선택

  4. Step name: Extract ZIP 의미있는 이름으로 지정하자

  5. Working directory: bin/ 배포압축파일이 bin 폴더에 있다.

    - 폴더 모양을 눌러서 폴더 위치를 지정할 수있으니 참고 하자

  6. script content: unzip -O CP949 *.zip -d result

    -O CP949 는 한글 파일명칭을 인코딩하기 위함

    *.zip -d result 는 현재 위치의 모든 zip을 result 폴더에 압축 해제

만약 압축 해제시 기존 압축 푼 파일을 삭제 하고 싶다면
#!/bin/bash set -e DEST_DIR="result" ZIP_PATTERN="*.zip" if [ -d "$DEST_DIR" ]; then echo "디렉토리 '$DEST_DIR'가 존재합니다. 기존 파일을 삭제합니다." rm -rf "${DEST_DIR:?}/"* else echo "디렉토리 '$DEST_DIR'가 존재하지 않습니다. 생성합니다." mkdir "$DEST_DIR" fi echo "압축을 해제합니다: $ZIP_PATTERN -> $DEST_DIR" unzip -O CP949 $ZIP_PATTERN -d $DEST_DIR echo "압축 해제가 완료되었습니다."
