Unity 이동_회전_위치

Transform.position : 객체를 즉시 이동 시키고 싶을때 사용

MoveTowards 또는 Rigidbody.MovePosition : 객체를 특정 위치로 일정한 속도로 이동시키고 싶을 때 사용

Translate : 객체를 방향 벡터를 기반으로 이동시키고 싶을 때 사용

위치

  • Transform.position

- 설명: 객체의 월드 좌표계에서의 위치를 나타내는 속성입니다. - 사용 방법: 위치를 직접 설정하거나 가져올 때 사용합니다. - 사용 시기: 객체를 특정 위치로 즉시 이동시킬 때 사용합니다. - 예시: transform.position = new Vector3(0, 5, 0); Vector3 currentPosition = transform.position;

  • Vector3.MoveTowards

- 설명: 두 점 사이를 일정한 속도로 이동합니다. - 사용 방법: 이동하고자 하는 위치를 계산하여 transform.position에 할당합니다. - 사용 시기: 속도 기반 이동을 구현할 때 사용합니다. - 예시: transform.position = Vector3.MoveTowards(transform.position, targetPosition, speed * Time.deltaTime);

  • Rigidbody.MovePosition

- 설명: 물리 엔진을 통해 객체를 이동시킵니다. - 사용 방법: Rigidbody 컴포넌트가 부착된 객체에서 사용하며, 이동하고자 하는 위치를 전달합니다. - 사용 시기: 물리적인 상호 작용을 고려해야 하는 객체를 이동시킬 때 사용합니다. - 예시: void FixedUpdate() { Vector3 newPosition = Vector3.MoveTowards(rigidbody.position, targetPosition, speed * Time.fixedDeltaTime); rigidbody.MovePosition(newPosition); }

Transform.rotation 또는 transform.eulerAngles : 객체의 회전을 즉시 변경

Transform.Rotate : 연속적인 회전을 구현

Quaternion targetRotation : 목표 회전까지 일정한 속도로 회전

회전

  • Transform.rotation

- 설명: 객체의 회전을 나타내는 Quaternion 값입니다. - 사용 방법: 회전을 직접 설정하거나 가져올 때 사용합니다. - 사용 시기: 객체의 회전을 즉시 변경해야 할 때 사용합니다. - 예시: transform.rotation = Quaternion.Euler(0, 90, 0); Quaternion currentRotation = transform.rotation;

  • Transform.Rotate

- 설명: 객체를 지정된 축을 기준으로 회전시킵니다. - 사용 방법: 회전 축과 각도를 전달하여 회전합니다. - 사용 시기: 연속적인 회전을 구현할 때 사용합니다 - 예시: transform.Rotate(Vector3.up * rotationSpeed * Time.deltaTime);

  • Quaternion.RotateTowards

- 설명: 현재 회전에서 목표 회전까지 일정한 속도로 회전합니다. - 사용 방법: 새로운 회전 값을 계산하여 transform.rotation에 할당합니다. - 사용 시기: 속도 기반 회전을 구현할 때 사용합니다. - 예시: Quaternion targetRotation = Quaternion.Euler(0, 180, 0); transform.rotation = Quaternion.RotateTowards(transform.rotation, targetRotation, rotationSpeed * Time.deltaTime);

  • Transform.LookAt

- 설명: 객체가 특정 위치를 바라보도록 회전시킵니다. - 사용 방법: 바라볼 대상의 위치를 전달합니다. - 사용 시기: 객체가 항상 특정 대상을 향하도록 해야 할 때 사용합니다. - 예시: transform.LookAt(targetPosition);
이동

  • Transform.Translate

- 설명: 객체를 특정 방향으로 이동시킵니다. - 사용 방법: 이동할 방향과 크기를 전달합니다. - 사용 시기: 방향 기반 이동을 구현할 때 사용합니다. - 예시: transform.Translate(Vector3.forward * speed * Time.deltaTime);

  • CharacterController.Move

- 설명: 캐릭터 컨트롤러를 사용하여 객체를 이동시킵니다. - 사용 방법: 이동할 벡터를 전달합니다. - 사용 시기: 캐릭터의 이동을 물리 엔진 없이 구현할 때 사용합니다. - 예시: CharacterController controller = GetComponent<CharacterController>(); Vector3 move = Vector3.forward * speed * Time.deltaTime; controller.Move(move);

  • NavMeshAgent.SetDestination

- 설명: AI 에이전트가 목표 지점까지 경로를 찾아 이동하도록 합니다. - 사용 방법: 목표 위치를 전달합니다. - 사용 시기: 경로 찾기(Pathfinding) 기능이 필요한 경우 사용합니다. - 예시: NavMeshAgent agent = GetComponent<NavMeshAgent>(); agent.SetDestination(targetPosition);

로컬 좌표계(Local Space):

- 객체 자신의 좌표계를 의미합니다.

- 객체의 방향과 회전에 따라 축이 결정됩니다.

월드 좌표계(World Space):

- 전체 장면의 전역 좌표계를 의미합니다.

- 축이 고정되어 있습니다.

사용 방법:

- transform.Translate나 transform.Rotate 함수에서 Space.Self(기본값)나 Space.World를 지정하여 좌표계를 선택할 수 있습니다.

// 로컬 좌표계에서 이동 transform.Translate(Vector3.forward * speed * Time.deltaTime, Space.Self); // 월드 좌표계에서 이동 transform.Translate(Vector3.forward * speed * Time.deltaTime, Space.World);
