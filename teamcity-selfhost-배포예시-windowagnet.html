    3.bulid step

    Advanced options 선택하면 작업 폴더 설정은 기준경로가 된다.(bin)

    1. PowerShell을 선택한다.

    2. Script: Source Code 선택한다.

    3. 결과물은 기본폴더에 있지만 효율적인 작업을 위해 다른 폴더에 결과물을 만든다.

      # PowerShell 스크립트 시작 $ErrorActionPreference = "Stop" $DEST_DIR = "result" $ZIP_PATTERN = "*.zip" if (Test-Path -Path $DEST_DIR) { Write-Host "디렉토리 '$DEST_DIR'가 존재합니다. 기존 파일을 삭제합니다." Remove-Item -Path "$DEST_DIR\*" -Recurse -Force } else { Write-Host "디렉토리 '$DEST_DIR'가 존재하지 않습니다. 생성합니다." New-Item -Path $DEST_DIR -ItemType Directory | Out-Null } Write-Host "압축을 해제합니다: $ZIP_PATTERN -> $DEST_DIR" Get-ChildItem -Path $ZIP_PATTERN | ForEach-Object { Expand-Archive -Path $_.FullName -DestinationPath $DEST_DIR -Force } Write-Host "압축 해제가 완료되었습니다."

    3.bulid step

    1. PowerShell을 선택한다.

    2. 작업 경로 bin

    3. 결과물은 기본폴더에 있지만 효율적인 작업을 위해 다른 폴더에 결과물을 만든다.

      # PowerShell 스크립트 시작 # 오류 발생 시 스크립트 중단 $ErrorActionPreference = "Stop" # 디버깅을 위해 실행 과정을 출력 $VerbosePreference = "Continue" # 환경 변수에서 GitHub 토큰 가져오기 $GIT_TOKEN = $env:GIT_TOKEN $GIT_USER = "bigcat0815" $GIT_EMAIL = "bigcat0815@gmail.com" $GIT_REPO = "https://${GIT_USER}:${GIT_TOKEN}@github.com/bigcat0815/yesbigcat.github.io.git" $TARGET_BRANCH = "main" # Git 설정 git config --global user.name "$GIT_USER" git config --global user.email "$GIT_EMAIL" # 현재 디렉토리 확인 Write-Host "현재 디렉토리: $(Get-Location)" # 'site' 디렉토리가 존재하면 삭제 if (Test-Path -Path "site") { Write-Host "'site' 디렉토리가 이미 존재합니다. 삭제합니다." Remove-Item -Path "site" -Recurse -Force } # main 브랜치 클론 Write-Host "GitHub 저장소를 '$TARGET_BRANCH' 브랜치로 클론합니다." git clone --branch $TARGET_BRANCH --single-branch $GIT_REPO site if ($LASTEXITCODE -ne 0) { Write-Error "Git clone failed" exit 1 } Set-Location -Path "site" # 클론된 저장소에 파일이 있는지 확인 if ((Get-ChildItem -Path . | Measure-Object).Count -gt 0) { Write-Host "클론된 저장소에 파일이 존재합니다." } else { Write-Error "클론된 저장소가 비어있습니다. 초기 커밋을 확인하세요." exit 1 } # 기존 파일 삭제 Write-Host "기존 파일을 삭제합니다." git rm -rf . if ($LASTEXITCODE -ne 0) { Write-Host "삭제할 파일이 없습니다." } # 압축 해제된 파일 복사 Write-Host "압축 해제된 파일을 복사합니다." Copy-Item -Path "..\result\*" -Destination "." -Recurse -Force # 변경 사항 추가 Write-Host "변경 사항을 Git에 추가합니다." git add . # 변경 사항이 있는지 확인 $changes = git diff --cached --exit-code if ($LASTEXITCODE -ne 0) { Write-Host "변경 사항이 있습니다. 커밋하고 푸시합니다." git commit -m "Deploy to GitHub Pages via TeamCity build" git push origin $TARGET_BRANCH Write-Host "배포가 성공적으로 완료되었습니다." } else { Write-Host "변경 사항이 없습니다. 배포할 내용이 없습니다." }
