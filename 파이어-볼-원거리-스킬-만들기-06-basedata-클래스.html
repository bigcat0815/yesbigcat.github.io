6-1. BaseData Class 설계

filename : Data.Contents.cs

    BaseData Class는 모든 Data의 부모이다.

    • BaseData

      public class BaseData { public int TemplateId; public string InternalName; public string NameTextId; public string DescriptionTextID; public string IconImage; public string PrefabName; }

    Data는 직렬화/역직렬화/게임내에서 Data로 사용되는 Class

    DataLoader JSON 파일을 로드하여 객체로 역직렬화한 후, 인게임 시작시 데이터의 유효성을 검사하고 다른 데이터와의 관계를 설정하는 중요한 역할

    6-2. JSON 역직렬화 위한 인터페이스

    filename : DataManager.cs

      1. IValidate : 인게임시 필요한 데이터를 JSON으로 부터 로드하여 캐쉬

        2. public interface IValidate { bool Validate(); }

      2. ILoader : MakeDict은 JSON을 인게임 메모리를 캐쉬하기 위한 헬퍼인터페이스

      public interface ILoader<TKey, TValue> : IValidate { Dictionary<TKey, TValue> MakeDict(); }

      MakeDict은 CSV->JSON 변환 과정과 무관
