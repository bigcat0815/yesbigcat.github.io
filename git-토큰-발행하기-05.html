Git 토큰 발행하기 05

Teamcity에서 githubpage로 push하기 위해 계정 토큰이 필요하다.

5. Git 토큰 발급

  1. 계정 settings | Developer settings | Personal access tokens | Tokens (classic) 이동

  2. Generate new token

  3. Note : 토큰 이름 지정

  4. Expiration : 토큰의 유효기간 설정을 넉넉하게 하자

  5. Select scopes : 토큰이 git에서 사용할 수 있는 권한을 repo 선택, 저장소를 의미한다.

토큰은 한 번 발급후 닫으면 안보이니 잠시 메모장에 기록해 놓고 사용 후 메모는 지우자.
