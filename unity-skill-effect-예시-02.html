Unity Skill과 Effect 및 투사체 스킬 예시 02

Skill

  1. 정의 :

    캐릭터(Hero)가 능동적으로 사용하는 능력이나 기술

    게임 내에서 특정 행동을 수행하기 위한 명령이나 기능

  2. 게임내의 스킬사용 :

    얼음 화살 발사

    Hero가 슬라임에게 얼음 화살을 발사

  3. 역할 :

    사용 조건 관리: 스킬 사용에 필요한 마나, 쿨다운 등을 관리

    대상과 범위 결정: 스킬이 어떤 대상에게 어떻게 적용될지 결정

    이펙트 및 투사체 생성: 스킬 사용 시 발생하는 이펙트나 투사체를 생성

Projectile

  1. 정의 :

    투사체는 스킬 사용 시 생성되어 게임 세계를 이동하는 물리적인 객체

    이는 스킬의 효과를 대상에게 전달하는 역할

    Prefab 이다

  2. 투사체(prefab) 사용 :

    Hero가 발사한 얼음 화살이 투사체

    슬라임을 향해 이동하여 스킬의 효과를 전달

  3. 역할 :

    이동 및 충돌 처리: 투사체는 특정 속도로 이동하며, 대상과의 충돌 여부를 판단

    이펙트 전달: 투사체가 대상에 도달하거나 충돌할 경우, 스킬의 효과를 적용

    시각적 연출: 플레이어에게 스킬 사용의 결과를 시각적으로 보여줌

Effect

  1. 정의 :

    스킬이나 투사체로 인해 대상에게 적용되는 상태 변화나 효과

    대상의 능력치나 상태를 변경

  2. 피해 이펙트(즉시) 1:

    15의 데미지: 슬라임이 얼음 화살에 맞아 받는 피해

    즉각적인 체력 감소: 슬라임의 체력이 15만큼 감소

  3. 상태 이상 이펙트(지속) 2:

    이동 속도 50% 감소: 슬라임은 2초 동안 이동 속도가 절반으로 감소

    디버프 효과: 슬라임에게 적용되는 부정적인 상태 변화

  4. 역할 :

    상태 변화 적용: 대상의 능력치(체력, 이동 속도 등)를 변경

    지속 시간 관리: 이펙트의 지속 시간(여기서는 2초)을 관리하며, 시간이 지나면 효과를 제거

    효과 누적 및 중첩 관리: 여러 이펙트가 동시에 적용될 경우 그 영향을 관리

얼음 폭발

스킬 : 범위 스킬 사용시 주변적에게 둔화와 피해 이펙트 적용

투사체 : 즉시 발동되는 범위 스킬이므로 투사체 불필요

피해 이펙트 : 범위 내의 모든 적들이 즉시 피해

둔화 이펙트: 적들의 이동 속도가 일정 시간 동안 지속 감소

VFX(시각적효과) 참고

  • 특징: 그래픽 연출 중심, 게임 플레이(effect)에 직접 영향 없음

  • 예시: "궁수가 화살 스킬을 사용하여 적을 공격한다."

  • 화살 모델과 애니메이션: 화살이 활에서 발사되어 날아가는 그래픽.

  • 타격 효과: 화살이 적에게 맞았을 때 나타나는 충돌 애니메이션.

  • 잔상 효과: 화살이 날아가는 경로에 나타나는 시각적 효과.

구분

시각적 효과 (VFX)

게임플레이 이펙트 (Gameplay Effects)

목적

플레이어에게 시각적인 피드백 제공

게임 내 메커니즘에 영향을 주는 상태 변화 적용

특징

그래픽 연출 중심, 게임 플레이에 직접 영향 없음

데이터 및 로직 중심, 캐릭터의 능력치나 상태 변경

영향

범위 플레이어의 감각적 경험 향상

캐릭터의 능력치, 상태, 행동 등에 직접적인 영향

예시

파티클 효과, 애니메이션, 광원 효과 등

데미지, 버프/디버프, 상태 이상 등

구현

방식 그래픽 엔진이나 시각 효과 시스템

게임 로직, 상태 관리 시스템
