Unbutu openssh 서버 설치

1. Unbuntu 수동 설치

24.04버전에서는 옵션체크시 설치가능

  1. Openssh 설치

    sudo apt update sudo apt install openssh-server

  2. 서비스 시작

    sudo systemctl status ssh sudo systemctl enable ssh sudo systemctl start ssh

2. Unbuntu ssh 서버 port변경

기본 22포트 사용은 최대한 피하자

  1. 유용한 툴설치

    systemctl status ssh sudo apt install net-tools

  2. 포트 확인

    netstat -an|grep LIST sudo systemctl status ssh

  3. sshd_config 원하는 포트번호 변경

    sudo vi /etc/ssh/sshd_config Port 22222

  4. 서비스 재시작

    sudo systemctl stop ssh.socket sudo systemctl stop ssh sudo systemctl daemon-reload sudo systemctl start ssh sudo systemctl status ssh

  5. 변경된 포트 서버에서 허용

    sudo ufw enable sudo ufw allow 22222

  6. ssh 서비스 재시작

    sudo systemctl stop ssh sudo systemctl start ssh

외부 접속 허용 하려면 공유기 포트포워딩 필수

3. ssh 자동 재시작 설정

  1. 자동 실행할 .sh 파일 만들기

    sudo vi /etc/init.d/autostart.sh

  2. 내용 기입

    #!/bin/bash nohup systemctl start ssh & sh /home/ubuntu/start-node-1.sh

    nohup 백그라운드 실행 // .sh 실행하려면 sh 명령어를 사용해야한다.

  3. 파일 권한

    sudo chmod 755 /etc/init.d/autostart.sh

  4. 부팅시 스크립트 실행하도록 rc.d에 등록

    update-rc.d autostart.sh defaults
