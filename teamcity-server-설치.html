TeamCity Server 설치

0. 기존 TeamCity 삭제

  • 설치한 경로

    sudo systemctl stop teamcity sudo systemctl disable teamcity sudo rm /etc/systemd/system/teamcity.service sudo systemctl daemon-reload sudo rm -rf /opt/teamcity

1. TeamCity 설치전 준비

  1. teamcity 계정 만들기

    su root adduser teamcity

  2. wget 설치

    apt update apt install wget -y

  3. Java 설치

    sudo apt update sudo apt install openjdk-17-jdk echo $JAVA_HOME java -version

1. Teamcity Server 설치

  1. 설치할 경로 만들기

    mkdir -p /opt/TeamCity/ cd /opt/TeamCity/

  2. teamcity 파일 받기/압축해제

    최신버전 확인https://www.jetbrains.com/teamcity/download/other.html

    wget https://download.jetbrains.com/teamcity/TeamCity-2024.07.3.tar.gz tar xfvz TeamCity-2024.07.3.tar.gz

  3. 계정 확인및 소유권 변경

    cut -d: -f1 /etc/passwd chown -R teamcity:teamcity /opt/TeamCity su teamcity

  4. 실행

    TeamCity/bin/runAll.sh start

2. Teamcity Server 자동재시작

  1. root 계정전환

    su root

  2. 서비스 등록

    vim /etc/systemd/system/teamcity.service

  3. 서비스 편집

    [Unit] Description=TeamCity Server After=network.target [Service] Type=forking User=teamcity Group=teamcity # TeamCity 설치 경로에 맞게 수정 ExecStart=/opt/TeamCity/TeamCity/bin/runAll.sh start ExecStop=/opt/TeamCity/TeamCity/bin/runAll.sh stop # 서비스가 실패할 경우 재시작 설정 Restart=on-failure RestartSec=10 [Install] WantedBy=multi-user.target

  4. 포트 오픈

    ufw allow 8111/tcp

  5. teamcity로 계전전환

    su teamcity

  6. 서비스 재시작

    systemctl daemon-reload systemctl restart teamcity systemctl enable teamcity systemctl status teamcity
