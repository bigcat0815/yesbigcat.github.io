Unity LifeCycle

게임 오브젝트 생성시

  1. Awake()

  • 호출 시점 : 게임 오브젝트가 생성 될때 1회 호출

  • 역할 : 자신의 오브젝트의 컴포넌트들 참조, 변수 초기화

  1. OnEnable()

  • 호출 시점 : 게임 오브젝트가 활성화 될때 마다 호출

  • 역할 : 이벤트 구독, UI Refresh 초기화 작업

  1. Start()

  • 호출 시점 : 게임 오브젝트가 생성되고 Awake()와 OnEnable()이 모두 호출된 후 1회 호출

  • 역할 : 다른 오브젝트,컴포넌트를 참조할 수 있다.

  1. Update(), FixedUpdate(), LateUpdate()

  • 호출 시점 : 게임 루프의 각 프레임마다 반복적으로 호출

  • 역할 : 사용자 입력 처리, 게임 로직 업데이트, 물리 계산 등.

게임 오브젝트 비활성화 및 파괴시

  1. OnDisable()

  • 호출 시점 : 게임 오브젝트가 비활성화시 마다 호출 또는 활성화시 파괴 직전일때 호출

  • 역할 : 이벤트 구독 해제, 리소스 정리

  1. OnDestroy()

  • 호출 시점 : 게임 오브젝트가 완전히 파괴되기 직전 1회 호출

  • 역할 : 최종 정리 작업, 메모리 해제

각 생명주기는 부모가 먼저 호출 된다.

부모 Awake -> 자식 Awake -> 부모 OnEnable -> 자식 OnEnable

base.Awake()와 같은 부모 함수를 먼저 호출하자
