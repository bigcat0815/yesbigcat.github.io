Writerside와 git 연동 02

2. Writerside 프로젝트용 Git 연동

  1. Writerside 프로젝트(DocuTech)를 생성한다.

  2. GitHub에서 Writerside 프로젝트를 관리할 git을 생성한다.

  3. 생성된 git을 특정폴더(writerableProject)에 clone 한다

  4. 기존 writerside 프로젝트 폴더를 복사한다.(기존것 불필요시 삭제)

  5. git과 연동이 되어 commit / push 를 할 수 있게 되었다.
