Unity Skill과 Effect 및 근거리 스킬 예시 03

Skill

  1. 정의 :

    캐릭터(Hero)가 능동적으로 사용하는 능력이나 기술

    게임 내에서 특정 행동을 수행하기 위한 명령이나 기능

  2. 게임내의 스킬사용 :

    검의 일격 사용

    Hero가 슬라임에게 검의 일격을 휘두른다

  3. 역할 :

    사용 조건 관리: 스킬 사용에 필요한 마나, 쿨다운 등을 관리

    대상과 범위 결정: 스킬이 어떤 대상에게 어떻게 적용될지 결정

    이펙트 생성: 스킬 사용 시 발생하는 이펙트 생성

Projectile

  • 근접스킬은 투사체 불필요

Effect

  1. 정의 :

    스킬이나 투사체로 인해 대상에게 적용되는 상태 변화나 효과

    대상의 능력치나 상태를 변경

  2. 피해 이펙트(즉시) 1:

    20의 데미지: 슬라임이 검의 일격에 맞아 받는 피해

    즉각적인 체력 감소: 슬라임의 체력이 20만큼 감소

  3. 상태 이상 이펙트(지속) 2:

    5초 동안 지속 피해: 슬라임은 5초 동안 매초 2의 추가 피해

    디버프 효과: 슬라임에게 적용되는 부정적인 상태 변화

  4. 역할 :

    상태 변화 적용: 대상의 능력치(체력, 이동 속도 등)를 변경

    지속 시간 관리: 이펙트의 지속 시간(여기서는 2초)을 관리하며, 시간이 지나면 효과를 제거

    효과 누적 및 중첩 관리: 여러 이펙트가 동시에 적용될 경우 그 영향을 관리

파이어볼

스킬 : 투사체 스킬 사용시 즉시 피해와 지속 이펙트 적용

투사체 : 대상에게 날아가야 하므로 투사체 필요

피해 이펙트 : 대상은 30의 데미지를 즉시 피해

둔화 이펙트: 대상은 3초 동안 매초 5의 지속 피해
