Unbuntu 명령어

1. Unbuntu 필수 명령어

  1. sudo apt update

    패키지 목록 업데이트

  2. sudo apt upgrade

    설치된 패키지 업그레이드

  3. sudo apt install openssh-server

    패키지 설치

  4. sudo systemctl status ssh

    서비스 확인

  5. sudo systemctl enable ssh

    재부팅시 자동 실행

  6. sudo systemctl start ssh

    서비스 바로 시작

  7. sudo systemctl daemon-reload

    설정이 변경시 시스템에 반영

  8. sudo ufw enable sudo ufw allow 7777

    포트번호 open

2. Unbuntu 생성/삭제 명령어

  1. pwd

    현재 폴더 위치

  2. mkdir myfolder

    폴더 생성

  3. mkdir -p /home/alice/projects/myproject

    경로에 없는 폴더 생성

  4. rmdir myfolder

    빈 폴더만 삭제 가능

  5. rm -rf myfolder

    경고 없이 강제 삭제

3. Unbuntu 사용자 확인 명령어

사용자명❌사용자ID:그룹ID:사용자 설명:홈 디렉토리:로그인 셸

  1. who

    현재 로그인 사용자 확인

  2. cat /etc/passwd

    사용자 목록 확인

  3. cat /etc/group

    그룹 목록 확인

  4. cut -d: -f1 /etc/passwd

    필요한 cell 잘라서 보기

  5. chmod -R 755 /myfolder

    소유자는 rwx(7) / 소유자 그룹(rx) / 다른사용자(rx)

권한 : r(읽기) =4 , w(쓰기) = 2, x(실행) = 1

4. Unbuntu 권한 명령어

  1. chmod 755 /myfolder

    폴더 소유자는 읽기/쓰기/실행, 그룹,다른사용자는 읽기/실행

  2. schown -R bigcat:developers /myfolder

    bigcat과 developers 그룹은 해당 폴더의 소유자가 된다.

  3. chmod -R 755 /myfolder

    소유자는 rwx(7) / 소유자 그룹(rx) / 다른사용자(rx)

-x : 압축해제 , -v : 과정출력 , -f: 파일이름 ,-z : gz,tgz , -j : bz2,bzip2

5. Unbuntu weget,uzip,tar

  1. wget http://jetbrains/update/buildAgent.zip

    해당 주소에서 파일 다운로드

  2. unzip buildAgent.zip -d /opt/teamcity-agent

    해당 폴더로 압축해제

  3. tar -xzvf myarchive.tar.gz -C /path/to/directory

    -z옵션 gzip를 특정폴더 압축해제

  4. tar -xjvf myarchive.tar.bz2

    -j옵션 bz2 압축해제
