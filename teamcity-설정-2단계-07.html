TeamCity 설정 2단계 07

7. TeamCity Job 설정 2

  1. Job 1 → step2 → SCRIPT 선택

  2. Step name: Deploy to github-pages

  3. Working directory: bin/ 에서 result를 찾아서 githubpage에 배포함

  4. script는 아래 기술

Step2 script

  1. GIT_TOKEN → 다음 단계에 설명

  2. GIT_USER → 계정 ID

  3. GIT_REPO → @github.com/계정/githubpage프로젝트의 git 주소

  4. cp -R ../result/* . 작업 폴더위치가 안맞으면 빌드 fail이 된다.

  5. Working directory: bin/ 에서 result를 찾아서 githubpage에 배포함

  6. 아래 스크립트 : step2 script

step2 script
#!/bin/bash set -e # 디버깅을 위해 실행 과정을 출력 set -x # 환경 변수에서 GitHub 토큰 가져오기 GIT_TOKEN=%env.GIT_TOKEN% GIT_USER=bigcat0815 GIT_EMAIL=bigcat0815@gmail.com GIT_REPO=https://$GIT_USER:$GIT_TOKEN@github.com/bigcat0815/bigcat0815.github.io.git TARGET_BRANCH=main # Git 설정 git config --global user.name "$GIT_USER" git config --global user.email "$GIT_EMAIL" # 현재 디렉토리 확인 echo "현재 디렉토리: $(pwd)" # 'site' 디렉토리가 존재하면 삭제 if [ -d "site" ]; then echo "'site' 디렉토리가 이미 존재합니다. 삭제합니다." rm -rf site fi # main 브랜치 클론 echo "GitHub 저장소를 '$TARGET_BRANCH' 브랜치로 클론합니다." git clone --branch $TARGET_BRANCH --single-branch $GIT_REPO site || { echo "Git clone failed"; exit 1; } cd site # 클론된 저장소에 파일이 있는지 확인 if [ "$(ls -A)" ]; then echo "클론된 저장소에 파일이 존재합니다." else echo "클론된 저장소가 비어있습니다. 초기 커밋을 확인하세요." exit 1 fi # 기존 파일 삭제 echo "기존 파일을 삭제합니다." git rm -rf . || echo "삭제할 파일이 없습니다." # 압축 해제된 파일 복사 echo "압축 해제된 파일을 복사합니다." cp -R ../result/* . # 변경 사항 추가 echo "변경 사항을 Git에 추가합니다." git add . # 변경 사항이 있는지 확인 if ! git diff --cached --exit-code > /dev/null; then echo "변경 사항이 있습니다. 커밋하고 푸시합니다." git commit -m "Deploy to GitHub Pages via TeamCity build" git push origin $TARGET_BRANCH echo "배포가 성공적으로 완료되었습니다." else echo "변경 사항이 없습니다. 배포할 내용이 없습니다." fi
