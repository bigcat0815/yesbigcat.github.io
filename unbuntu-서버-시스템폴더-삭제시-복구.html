Unbuntu 서버 시스템폴더 삭제시 복구

Unbuntu 복구 설치

  1. /bin 폴더 삭제일때 심볼릭 링크 연결

    ls -l / sudo ln -s /usr/bin /bin

  2. coreutils 패키지를 재설치 하여 핵심 유틸리티 설치

    sudo apt update sudo apt install --reinstall coreutils

  3. 시스템의 기본 패키지를 다시 설치

    sudo apt install --reinstall ubuntu-minimal

이 방법이 안된다면 시스템 완전 재설치 필요하다
