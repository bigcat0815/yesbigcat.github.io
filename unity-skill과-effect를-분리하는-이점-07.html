Unity Skill과 Effect를 분리하는 이점 07

게임 밸런싱에 유리

  1. 개별 조정 가능: 밸런스 조절의 용이성

    - 상황: 플레이어들이 '빙결 화살' 스킬이 너무 강력하다고 피드백

    - 대응1: 이펙트 모듈에서 '빙결' 이펙트의 지속 시간을 3초에서 2초로 감소

    - 대응2: 또는 스킬 모듈에서 스킬의 재사용 대기 시간을 늘립

    - 이점1 : 스킬의 각 요소를 독립적으로 조정

    - 이점2 : 수정 사항이 즉시 게임에 적용

  2. 데이터 주도 개발: 손쉬운 파라미터 변경

    - 이펙트의 수치(피해량, 지속 시간 등)를 코드가 아닌 데이터 파일로 관리

    - 이점1 : 게임 디자이너나 밸런서가 직접 데이터를 수정하여 게임 플레이를 조정

    - 이점2 : 게임을 재빌드하지 않고도 데이터 변경만으로 수정 사항을 반영

    - 이점3 : 수치 변경이 용이하여 다양한 실험과 조정이 가능

    - 이점4 : 데이터 파일의 변경 내역을 추적하여 어떤 수정이 있었는지 확인
